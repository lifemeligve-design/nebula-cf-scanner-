## 📖 معرفی

**اسکنر فضایی آی‌پی تمیز کلودفلر** یک ابزار حرفه‌ای و گرافیکی برای پیدا کردن آی‌پی‌های تمیز و پایدارِ شبکه‌ی Cloudflare است؛ مخصوص کاربرانی که از V2Ray / Cloudflare Workers روی اپراتورهای ایرانی (همراه اول، ایرانسل، شاتل و…) استفاده می‌کنند.

برنامه به‌صورت خودکار آی‌پی‌های کلودفلر را پیدا، تست و رتبه‌بندی می‌کند و دقیقاً مشخص می‌کند هر آی‌پی برای کدام اپراتور تمیز است.

---

## ✨ امکانات

- 🔍 **تأیید چنددوره‌ای دقیق** — هر آی‌پی با چند دور TCP + TLS + WebSocket + بررسی cf-ray تست می‌شود
- 📱 **پروفایل اپراتورها** — پروفایل بهینه برای همراه اول و ایرانسل (شبکه‌های پرنوسان موبایل)
- 🏷️ **برچسب اپراتور** — مشخص می‌کند هر آی‌پی برای کدام اپراتور(ها) تمیز است
- 🧠 **اسکن یادگیرنده و هوشمند /24** — تمرکز روی رنج‌های پربازده برای سرعت و دقت بیشتر
- 🌍 **پشتیبانی IPv6 + تست سرعت دانلود واقعی**
- 🛡️ **حالت اینترنت ملی** — تشخیص وضعیت اینترنت + پیدا کردن آی‌پی‌های بازمانده
- 📊 **داشبورد تحلیلی زنده + استخر تاریخی آی‌پی‌ها**
- 🚀 **خروجی برای v2rayNG / Clash / sing-box / Hiddify**
- 🎨 **رابط فضایی انیمیشنی** با دکمه‌های نئونی و چیدمان ریسپانسیو

## ⬇️ دانلود و نصب

فایل اجرایی آماده را از بخش **[Releases](../../releases)** دانلود کنید.

1. به صفحه‌ی [Releases](../../releases) بروید
2. آخرین نسخه‌ی `CFCleanScanner.exe` را دانلود کنید
3. روی فایل دوبار کلیک کنید — **بدون نیاز به نصب پایتون**

> 💡 اگر آنتی‌ویروس هشدار داد، طبیعی است (هشدار اشتباهِ معروفِ فایل‌های PyInstaller). فایل را در استثناها بگذارید.

## 🚀 راهنمای سریع استفاده

1. دامنه‌ی **SNI** خود را از کانفیگ V2Ray در فیلد مربوطه وارد کنید
2. **مسیر CDN (path)** را وارد کنید
3. برای موبایل، دکمه‌ی **«آی‌پی تمیز موبایل»** را بزنید و اپراتورتان را انتخاب کنید
4. یا دکمه‌ی **«شروع اسکن دقیق کلودفلر»** را بزنید
5. آی‌پی‌های تمیز را از لیست کپی کنید یا خروجی Subscription بگیرید
